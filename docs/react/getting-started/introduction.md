# Introduction

**Ez Scrollbar** is a small library that gives you custom scrollbars in your React app. If you've ever wanted scrollbars that match your design (or that work the same everywhere), this library is for you.

## Why use a custom scrollbar?

- **Consistent look** — Native scrollbars look different in Chrome, Firefox, Safari, and on Windows vs macOS. With Ez Scrollbar, you control the appearance.
- **Design match** — Style the track and thumb to fit your app (colors, size, rounded corners, etc.).
- **Same API everywhere** — One component works across browsers and platforms.

## Who is this for?

Ez Scrollbar is for you if:

- You're building a React app and want scrollbars that look good and behave the same everywhere.
- You're new to React and want something that "just works" with a simple `<ScrollBar>` wrapper.
- You need more control later — you can use the provided hooks to build your own scrollbar UI.

You don't need to be an expert. The next page will walk you through installing and your first scrollable area.

## What you'll need

- A React 18+ project (Vite, Create React App, or any other setup).
- A few minutes to install and try the basic example.

Ready? Head to [Installation](/react/getting-started/installation) to add the package and wrap your first scrollable content.
